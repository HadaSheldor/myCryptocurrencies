שמריהו הדס
----------

- ממשק המשתמש נראה טוב ונוח לשימוש.

- קבצי התמונות נמצאים כולם בתיקייה ייעודית לכך.

- קיימת הפרדה של קוד ה-CSS לקובץ חיצוני.

- קיימת הפרדה של קוד ה-JavaScript לקובץ חיצוני.

- ישנם שמות משמעותיים למשתנים ולפונקציות.

- ישנה שמירה על מוסכמת שמות נכונה עבור משתנים ופונקציות.

- ישנו תיעוד מסוים המאפשר הבנה טובה יותר של הקוד אך ממש מעט. חשוב לתעד מקומות מורכבים בקוד.

- אחלה שימוש ב-jQuery.

- ישנו שימוש טוב עם AJAX.

- ישנה קבלה והצגת JSON כמו שצריך.

- ישנה הצגה של המטבעות כמו שצריך.

- ניתן לראות מידע ספציפי בנוגע לכל מטבע ע"י לחצן ה-More Info.

- המידע של More Info נשמר לוקלית וניתן להצגה חוזרת מבלי לגשת שוב לשרת עבורו.

- ישנה הצגת הגרף כמו שצריך.

- ישנה הצגת Dialog לביטול מטבעות בבחירת מטבע שישי.

- ישנו דף About.

- מנגנון החיפוש עובד כמו שצריך. אחלה ביצוע חיפוש תוך כדי הקלדה!

- ישנו שימוש טוב ב-Bootstrap - מעולה.

- קיימת אסתטיקה בקוד מבחינת רווחים, Enter-ים, Indentations וכדומה. מעולה.

- ה-Console נקי משגיאות.

- האתר  בנוי כ-SPA מלא (Single Page Application). מעולה!

- ישנן פקודות CSS שנמצאות ב-HTML במקום שיהיו בקובץ css חיצוני בלבד. כל פקודות ה-css צריכות להיות בקובץ css חיצוני בלבד.

- קיימות הפניות משוכפלות לסקריפטים (כלומר ה-HTML מבקש את אותו הסקריפט מספר פעמים) שזה כמובן מיותר ויכול לגרום לבאגים.

- קוד ה-JavaScript עמוס מאוד. נכון היה לחלק את הקוד למספר קבצי JavaScript נפרדים.

- תיבת ה-Dialog מופיעה בסימון מטבע חמישי כאשר היא אמורה להופיע בסימון מטבע שישי בלבד (כי חמישה מטבעות זה חוקי).

- תיבת ה-Dialog לבחירת מטבע נוסף על החמישה גורמת לביטול בחירת מטבע אך לא גורמת לבחירת המטבע השישי שלחצו עליו.

- חסר יצירת IIFE העוטפת את כל הקוד או לחילופין שימוש ב-Document-Ready של jQuery על כל הקוד.
